# analizador-matematico-poo

# Características:
Paradigma POO: Utiliza clases para separar la lógica matemática de la interfaz gráfica.

Interfaz Gráfica (GUI): Construida con wxPython, permitiendo una interacción fluida.

Validación de Datos: Controla que el usuario ingrese únicamente números enteros positivos.

Cálculo Eficiente: Implementa algoritmos para encontrar divisores, determinar si es primo y mostrar la descomposición en factores.
